## Betaflight Tune

This Betaflight tune is optimized for the Memory Halo 4 frame

Tested on **Betaflight 4.5**.

To apply the CLI .txt file:

1. Connect to Betaflight Configurator
2. Open the **CLI**
3. Paste the contents of `memory_halo_4_tune.txt`
4. Press **Enter**
5. Type `save`

Your flight controller will reboot with the tune applied.

In case you want to double check if everything went correctly, or only adjust a few parameters, take a look at the [tune-screenshots](/tune-screenshots) folder
