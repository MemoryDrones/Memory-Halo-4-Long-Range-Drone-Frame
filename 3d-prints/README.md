## 3D Printed Parts

**Material:** TPU  
**Nozzle Temperature:** 205°C  
**Print Speed:** Default settings (20-30 mm/s recommended)

---

## Print Orientation

### Print by Object (Default)
All files should be printed **by object** for a clean aesthetic finish.

Includes: sideplates, camera mounts, antenna mounts, GPS mounts, etc.

### Print by Layer (Exceptions)
These files must be printed **by layer** to ensure proper fit and strength:

- `*_protective_arm_mount.stl`
- `*_arm_motor_mount.stl`
- `*_sma_antenna_mount.stl`
